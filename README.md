# Linux 入门教程 🐧

一个交互式的 Linux 命令行入门教程网页，帮助初学者快速掌握 Linux 基础知识。

## 🌟 特性

- 📱 **响应式设计** - 完美适配手机、平板和桌面设备
- 🎨 **现代化界面** - 渐变色彩和卡片式布局
- 💻 **代码高亮** - 清晰的命令行代码展示
- 📋 **一键复制** - 点击代码块即可复制
- ⚡ **无需构建** - 纯 HTML/CSS/JS，直接打开使用

## 📚 内容涵盖

1. **文件与目录操作** - `pwd`, `ls`, `cd`, `mkdir`, `touch`
2. **文件权限管理** - `chmod`, `chown`, 权限数字含义
3. **进程管理** - `ps`, `top`, `htop`, `kill`
4. **网络命令** - `ip`, `ping`, `netstat`, `curl`, `wget`
5. **文本处理** - `grep`, `sed`, `awk`, 管道操作
6. **包管理器** - `apt`, `dnf`, `pacman`

## 🚀 使用方法

### 本地浏览

直接在浏览器中打开 `index.html` 文件：

```bash
# 使用浏览器打开
firefox index.html
# 或
chrome index.html
```

### 部署到 GitHub Pages

1. Fork 本仓库
2. 进入 Settings → Pages
3. 选择 main 分支作为源
4. 访问 `https://yourusername.github.io/linux-tutorial`

## 🛠️ 本地开发

```bash
# 克隆仓库
git clone https://github.com/susurrune/linux-tutorial.git

# 进入目录
cd linux-tutorial

# 使用 Python 简单 HTTP 服务器预览
python3 -m http.server 8080

# 打开浏览器访问 http://localhost:8080
```

## 📖 学习路线

```
基础命令 → 权限管理 → 进程管理 → 网络工具 → 文本处理 → 包管理
   ↓           ↓           ↓           ↓           ↓           ↓
 ls/cd      chmod      ps/top      ping/curl    grep      apt/dnf
 mkdir      chown       kill       wget        sed/awk   pacman
 touch      rwx        htop       netstat      管道
```
