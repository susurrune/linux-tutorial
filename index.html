文件与目录操作

掌握 Linux 中最常用的文件和目录管理命令

# 查看当前目录
pwd

# 列出文件
ls -la

# 切换目录
cd /home/user

# 创建目录
mkdir myfolder

# 创建文件
touch file.txt
基础
🔐

文件权限管理

理解和修改 Linux 文件权限系统

# 查看文件权限
ls -l file.txt

# 修改权限 (r=4, w=2, x=1)
chmod 755 script.sh

# 添加可执行权限
chmod +x script.sh

# 修改所有者
chown user:group file.txt
进阶
⚙️

进程管理

查看和管理系统运行的进程

# 查看运行中的进程
ps aux

# 实时进程监控
top
htop

# 查找进程
pgrep process_name

# 结束进程
kill PID
killall process_name
系统
🌐

网络命令

网络诊断和连接管理工具

# 查看IP地址
ip addr
ifconfig

# 测试网络连通
ping google.com

# 查看端口连接
netstat -tuln
ss -tuln

# 下载文件
curl -O url
wget url
网络
📝

文本处理

强大的文本处理命令行工具

# 搜索文本
grep "pattern" file.txt

# 流编辑器
sed 's/old/new/g' file.txt

# 文本处理
awk '{print $1}' file.txt

# 管道组合
cat file.txt | grep "error" | wc -l
工具
📦

包管理器

不同发行版的软件包管理命令

# Debian/Ubuntu (apt)
apt update
apt install package
apt remove package

# CentOS/RHEL (yum/dnf)
dnf install package
yum install package

# Arch (pacman)
pacman -S package
